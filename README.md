# Asistente Personal 

Permite hacer consultas con voz a Paulina:

- Abrir YouTube"
- Abrir Google"
- Qué día es hoy"
- Qué hora es"
- Buscar en Wikipedia"
- Buscar en Internet"
- Reproducir en YouTube"
- Contar chiste"
- Repetir chiste"
- Precio de las acciones"
- Salir"

# Ejecucion:
```
pipenv shell

pipenv install

python3 main.py
```

### Consideraciones para Mac

Previamente instalar:

```
brew update
brew install portaudio
brew link --overwrite portaudio
pip install pyaudio
```
